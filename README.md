## Install Guide
1) Clone the repository into a local directory:
```
git clone git@github.com:tganshaw/senioritis__tudorg_amys90_hannah61_stevenw92.git senioritis
```
2) Enter the cloned repository directory:
```
cd senioritis
```
3) Create a virtual environment:
```
python3 -m venv venv
```
4) Activate the virtual environment for Linux, Windows, or Mac:

  a. Linux
  ```
  . venv/bin/activate
  ```
  b. Windows
  ```
  venv\Scripts\activate
  ```
  c. Mac
  ```
  source venv/bin/activate
  ```
5) Install necessary modules:
```
pip install -r requirements.txt
```
6) Change into the app directory:
```
cd app
```

## Launch codes
1) Run the app through Flask:
```
python3 __init__.py
```
2) Open the link:
```
https://senioritis.dev
```
3) After running the launch codes and utilizing the app, exit the virtual environment:
```
deactivate
```
